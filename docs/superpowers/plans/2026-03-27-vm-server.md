# 9P VmServer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a 9P FileServer that exposes VM lifecycle management (start/destroy/state) through `ctl` and `config` files, following the `GenetServer`/`VirtioNetServer` pattern.

**Architecture:** Command pattern — `HvServer` parses 9P writes into `VmCommand` values, the kernel executes them against the hypervisor externally. No trait, no bridge, no hypervisor reference in the server. Two new files in `harmony-microkernel`.

**Tech Stack:** Rust (no_std + alloc), existing `FileServer`/`FidTracker` from `harmony-microkernel`

**Spec:** `docs/superpowers/specs/2026-03-27-vm-server-design.md`

```rust
// crates/harmony-microkernel/src/hv_manager.rs
// SPDX-License-Identifier: GPL-2.0-or-later
//! VM lifecycle types for the 9P VmServer.
//!
//! The kernel writes commands to the VmServer's `ctl` file and reads
//! VM state from it. These types flow between the 9P server and the
//! kernel's hypervisor integration.

/// Commands produced by HvServer when the kernel writes to ctl.
#[derive(Debug, Clone, PartialEq, Eq)]
pub enum VmCommand {
    /// Start the VM. `entry_ipa` is the guest kernel entry point,
    /// `dtb_ipa` is the device tree blob address (written to x0 per ARM64 boot protocol).
    Start { entry_ipa: u64, dtb_ipa: u64 },
    /// Destroy the VM and free its resources.
    Destroy,
}

/// VM lifecycle state.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum VmState {
    Created,
    Running,
    Halted,
}

impl VmState {
    /// State as a human-readable string for the ctl file.
    pub fn as_str(&self) -> &'static str {
        match self {
            VmState::Created => "created\n",
            VmState::Running => "running\n",
            VmState::Halted => "halted\n",
        }
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn vm_state_as_str() {
        assert_eq!(VmState::Created.as_str(), "created\n");
        assert_eq!(VmState::Running.as_str(), "running\n");
        assert_eq!(VmState::Halted.as_str(), "halted\n");
    }

    #[test]
    fn vm_command_eq() {
        assert_eq!(
            VmCommand::Start { entry_ipa: 0x4000_0000, dtb_ipa: 0x4400_0000 },
            VmCommand::Start { entry_ipa: 0x4000_0000, dtb_ipa: 0x4400_0000 },
        );
        assert_ne!(VmCommand::Destroy, VmCommand::Start { entry_ipa: 0, dtb_ipa: 0 });
    }
}
```

## Task 2: HvServer — 9P FileServer

**Files:**
- Create: `crates/harmony-microkernel/src/hv_server.rs`
- Modify: `crates/harmony-microkernel/src/lib.rs`

**Context:**
- Follow `VirtioNetServer` pattern EXACTLY — reference `crates/harmony-microkernel/src/virtio_net_server.rs`
- **READ `virtio_net_server.rs` FIRST** — copy its structure method-by-method
- Not generic — `HvServer` is a concrete struct with `vmid: u8`, `mac: [u8; 6]`, `state: VmState`, `name: &'static str`, `tracker: FidTracker<()>`, `pending: Option<VmCommand>`
- QPaths: ROOT(0), DIR(1), CTL(2), CONFIG(3)
- Dynamic device name: `(QPATH_ROOT, n) if n == self.name`
- `ctl` read: `self.state.as_str()` via `slice_at_offset`
- `ctl` write: parse `"start 0x<entry_ipa> 0x<dtb_ipa>"` or `"destroy"`, store in `self.pending`
- `config` read: format `"vmid: N\nmac: xx:xx:xx:xx:xx:xx\n"` via `slice_at_offset`
- `take_pending_command(&mut self) -> Option<VmCommand>` — read and clear pending
- `update_state(&mut self, state: VmState)` — set current state

### Hex parsing for ctl write

```rust
fn parse_hex(s: &str) -> Option<u64> {
    let s = s.strip_prefix("0x").or_else(|| s.strip_prefix("0X")).unwrap_or(s);
    u64::from_str_radix(s, 16).ok()
}
```

### ctl write parsing

```rust
fn parse_ctl_command(data: &[u8]) -> Result<VmCommand, IpcError> {
    let s = core::str::from_utf8(data).map_err(|_| IpcError::InvalidArgument)?;
    let s = s.trim();
    if s == "destroy" {
        return Ok(VmCommand::Destroy);
    }
    if let Some(rest) = s.strip_prefix("start ") {
        let parts: Vec<&str> = rest.split_whitespace().collect();
        if parts.len() != 2 {
            return Err(IpcError::InvalidArgument);
        }
        let entry_ipa = parse_hex(parts[0]).ok_or(IpcError::InvalidArgument)?;
        let dtb_ipa = parse_hex(parts[1]).ok_or(IpcError::InvalidArgument)?;
        return Ok(VmCommand::Start { entry_ipa, dtb_ipa });
    }
    Err(IpcError::InvalidArgument)
}
```
